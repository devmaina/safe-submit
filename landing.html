Welcome to Safe-Submit App

Efficiently collect and manage user feedback from your site forms

What is Safe-Submit App?

Safe-Submit App is a powerful web application designed to streamline the process of collecting and managing feedback from your site forms. Built with cutting-edge technology, it ensures that all feedback is securely stored and easily accessible.

Our Tech Stack

The Safe-Submit App is built using modern and reliable technologies to ensure top performance and scalability:

  • Frontend: Vue.js
  • Backend: Flask
  • Database: PostgreSQL
  • Deployment: Docker & Nginx

How It Works

The Safe-Submit App integrates seamlessly with your existing site forms. When a user submits feedback, the data is securely sent to our application where it is stored in a PostgreSQL database. You can then view, filter, and manage the feedback through an intuitive web interface built with Vue.js.

Why Safe-Submit App?

Safe-Submit App provides a centralized solution for managing feedback, removing the hassle of dealing with mixed-up emails and hard-to-configure mail servers. With easy filtering by sender location, date sent, and more, you can quickly find and act on feedback. Plus, the app is mobile-friendly, allowing you to stay connected on the go.
